# WearWow 风格笔记（高保真 UI 对标参考）

参考截图见 `design-reference/wearwow/ww_1.jpg ~ ww_6.jpg`（来自 App Store 官方截图）。

## 整体感觉
干净、轻盈、有时尚杂志感。大量留白 + 白底卡片，靠服装照片本身出彩，UI 不抢戏。

## 配色
- 页面背景：极浅的柔和渐变（薄荷绿 → 浅粉 → 淡紫），或纯白/米白
- 卡片：纯白，大圆角
- 主文字：纯黑 / 近黑
- 强调色：黑色胶囊按钮 + 一处渐变彩色描边（粉→绿渐变，用在 AI 聊天输入框、Chat 按钮上，暗示"AI 能力"）
- AI 推荐卡（首页 Soft Power Suit 卡）：黑色底白字，与全页浅色形成强对比，是视觉锚点

## 字体排版
- 标题：超粗黑体（Heavy/Black 字重），字号很大
- 出现手写感/斜体元素（"Hi~"、"OOTD"、Playgrounds 卡上的斜体字）增加时尚感
- 正文灰色、字号小、行距松

## 组件语言
- 圆角非常大：卡片 ~20-24px，按钮全是胶囊形（full round）
- 分类筛选：圆形缩略图 + 右上角数量角标（Top / Bottoms / Dress / Outerwear）
- 快捷标签：浅灰胶囊 chips（Meeting / Wedding / Outdoors...）
- 底部导航 4 个 tab：Home / Wow AI / Wardrobe / Me，图标+文字，选中为黑色
- 悬浮操作：右下角黑色圆形 + 号，旁边"Free styling"胶囊
- 衣物图片：白底抠图（去背景），是整个产品的视觉基础
- 试穿卡片：照片斜置叠放、带阴影，像拍立得散落的感觉

## 关键页面结构（对照截图）
1. ww_3 首页：顶部 logo+天气 → 黑色 AI 推荐卡（左文右图）→ AI 聊天输入框（渐变描边）→ OOTD 一周日历横条 → Playgrounds 横滑卡 → Inspiration 瀑布流
2. ww_4 衣橱页：Item/Outfit 顶部切换 → 圆形分类筛选 → 两列白卡网格（衣物图 + 收藏数 + Style Now 按钮）→ 底部 tab
3. ww_5 穿搭日历：月视图，每天格子里放当天 outfit 的组合小图，可打勾
4. ww_1 AI 造型师聊天：场景 chips + 聊天输入
5. ww_2 / ww_6 灵感试穿：上传灵感图 → AI 生成上身效果，底部横滑切换不同 outfit

## 做高保真时的落地要点
- 手机宽度按 375~390px 设计，整页可竖向滚动
- 衣物素材用白底/浅底商品图，保持"抠图"观感
- 全局圆角、胶囊按钮、浅色渐变背景是最容易还原风格的三件事
- AI 相关入口统一用「粉绿渐变」点缀，其余保持黑白灰
